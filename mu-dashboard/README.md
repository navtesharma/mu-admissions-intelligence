# MU Admissions Intelligence Dashboard

Built for Masters' Union Founder's Office role application.

## Deploy to Vercel

1. Push this folder to a GitHub repo
2. Go to vercel.com → New Project → Import that repo
3. Framework: Next.js (auto-detected)
4. No environment variables needed
5. Deploy

## What's inside

- **Funnel Analytics** — 5-stage admissions funnel with drop-off quantification
- **Source Intelligence** — Channel CVR matrix showing where to invest
- **Profile Clusters** — Applicant segmentation vs placement outcomes
- **AI Analyst** — Claude-powered Q&A on your admissions data
- **Why This** — Problem statement, rationale, expansion vision
